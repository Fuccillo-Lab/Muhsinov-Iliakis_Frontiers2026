# Muhsinov-Iliakis_Frontiers2026
MATLAB scripts used to generate all analyses, Figures 2-4, and Supplemental Figures in Muhsinov et al. (2026) published in Frontiers in Cellular Neuroscience (doi: 10.3389/fncel.2026.1795921/).


## Scripts
**density_additional.m**: This code was used to generate supplemental figures in Muhsinov et al. (2026). 

**density_processing_and_plot.m**: This code was used to generate Figures 2 and 3 of Muhsinov et al. (2026).

**subregion_additional.m**: This code was used for analyses in Supplementary Figures related to partitioning of caudoputamen into four cortical input-defined clusters based on analysis of Hunnicutt et al. (2016). Relies on code from Tianyi Mao lab available on their GitHub under: https://github.com/TianyiMaoLab/striatumclusters. 

**subregion_processing_and_plot.m**: This code was used for analyses in Figure 4 and Supplementary Figures related to partitioning of caudoputamen into four cortical input-defined clusters based on analysis of Hunnicutt et al. (2016). Relies on code from Tianyi Mao lab available on their GitHub under: https://github.com/TianyiMaoLab/striatumclusters.


## Structure Masks from Allen CCFv3 (Wang et al., 2020)
The following masks from the Allen CCFv3 (Wang et al., 2020) were used to restrict analyses to the caudoputamen. These structure masks are available under the following link: https://download.alleninstitute.org/informatics-archive/current-release/mouse_ccf/annotation/ccf_2017/structure_masks/structure_masks_25/. 

**structure_672.nrrd**
This is the caudoputamen mask from Allen CCFv3. 

**Ventricle_Mask.nrrd**
This is the ventricle mask from Allen CCFv3, this was used to exclude periventricular PV+ ependymal cells.
